# Texts of Doom

Date: 1/11/24 - 2/11/24

Description: A game/prototype messaging applicaiton that only allows user to chat by defeating enemies who are the words they want to write.

### Requirements: 
- Node JS v24.11.0

### How to run:
To join a chat, go into the game folder within demo.html on line 253 and change the http://10.247.244.78 to the one outputted but the host of the session, the same needs to be done in line 63. After this you can run the join.bat file to chat with them.

To host a chat session run the start_host.bat function, this will display your local ipv4 address which if your attempting to chat with anyone they'll need. In addition within the chattingSystem.js in the chat folder the talkjs link on line 22 and the authorisation key will need to be changed to one of your own and the same process of changing the IP to your own as described above will need to occur on your own device.
